# Solver SAT - Rezoluție, DP și DPLL

Acest proiect conține implementările a trei algoritmi fundamentali pentru rezolvarea problemelor de satisfiabilitate booleană (SAT) reprezentate în formă normală conjunctivă (CNF). Proiectul a fost dezvoltat pentru a analiza eficiența diferitelor abordări de calcul în logica propozițională.

## Algoritmi Implementați
Proiectul include următoarele metode de rezolvare:

Rezoluție: O metodă bazată pe aplicarea succesivă a regulii rezoluției până la obținerea clauzei vide.

DP (Davis-Putnam): Algoritmul original bazat pe eliminarea variabilelor.

DPLL (Davis-Putnam-Logemann-Loveland): Un algoritm eficient bazat pe backtracking, unit propagation și literal elimination, care stă la baza solverelor moderne.
